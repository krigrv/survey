Survey Widget Demo Page

This is a demonstration page showing how the survey widget works when embedded in a website.

How it works:

  1. Create a form using the form builder
  2. Get the embed code from the form preview
  3. Paste the embed code into your website
  4. The survey will appear as a popup after a few seconds

Example Embed Code:

<script type="text/javascript" src="http://localhost:3001/popup/embed.js" data-form-id="YOUR_FORM_ID"></script>

The widget will automatically load and display your survey form as a popup overlay. Users can fill out the form and submit their responses, which will be saved to your database.

Note: To test the widget, you need to:

  • Create a form in the form builder
  • Copy the actual embed code with a real form ID
  • Replace the example embed code above with your actual code
